Web Development Introduction

Web development is the process of creating websites and web applications that people can access over the internet. At its core, it involves three main building blocks:

  • HTML (HyperText Markup Language), which structures the content of a web page;
  • CSS (Cascading Style Sheets), which styles the page and makes it visually appealing;
  • JavaScript, which adds interactivity and functionality, such as forms, animations, or dynamic content.

Together, these tools enable developers to build everything from simple blogs to complex platforms like social media sites. Web development also includes understanding how websites are hosted and accessed, ensuring they work smoothly on different devices and browsers.

What is HTML

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. At its heart, HTML is a language made up of elements, which can be applied to pieces of text to give them different meaning in a document (Is it a paragraph? Is it a bulleted list? Is it part of a table?), structure a document into logical sections (Does it have a header? Three columns of content? A navigation menu?), and embed content such as images and videos into a page.

<!DOCTYPE HTML>

What is JavaScript

JavaScript is a cross-platform, object-oriented scripting language used to make webpages interactive (e.g., having complex animations, clickable buttons, popup menus, etc.). There are also more advanced server-side versions of JavaScript such as Node.js, which allow you to add more functionality to a website than downloading files (such as real-time collaboration between multiple computers). Inside a host environment (for example, a web browser), JavaScript can be connected to the objects of its environment to provide programmatic control over them.

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements.

Data Types

The latest ECMAScript standard defines seven data types:

  • Boolean: true and false.
    • let isActive = true;
  • null: A special keyword denoting a null value. Because JavaScript is case-sensitive, null is not the same as Null, NULL, or any other variant.
    • let emptyValue = null;
  • undefined: A top-level property whose value is undefined.
    • let notDefined;
  • Number: 42 or 3.14159.
    • let age = 25;
  • String: "Howdy"
    • let name = "Alice";
  • Symbol (new in ECMAScript 2015): A data type whose instances are unique and immutable.
    • let sym = Symbol("description");
  • Object: key-value pairs where the keys (properties) are strings (or symbols), and the values can be any valid JavaScript data type, including other objects, functions, or arrays.
    • let person = { name: "John", age: 30, isActive: true };
